# Полный гайд по переносу системы
> Arch Linux / CachyOS + Hyprland + Silent SDDM

---

## Что сохранено в репо github.com/voxelpy/dotfiles

| Что | Где в репо |
|-----|-----------|
| Hyprland конфиг | `dot_config/hypr/` |
| Waybar | `dot_config/waybar/` |
| Wofi | `dot_config/wofi/` |
| Kitty | `dot_config/kitty/` |
| Nvim | `dot_config/nvim/` |
| Swaync | `dot_config/swaync/` |
| Waypaper | `dot_config/waypaper/` |
| Личные скрипты | `dot_local/bin/` |
| Silent SDDM тема | `sddm-theme-silent/` |
| SDDM конфиг | `sddm.conf` |
| SDDM Hyprland конфиг | `hyprland-sddm.conf` |
| Аватарка | `ivan.face.icon` |
| Все pacman пакеты | `pkglist.txt` |
| Все AUR пакеты | `aurlist.txt` |

---

## Одна команда для установки

### На своём ПК (Arch) или CachyOS:
```bash
bash <(curl -s https://raw.githubusercontent.com/voxelpy/dotfiles/main/install.sh)
```

### На ноуте (CachyOS) — то же самое, но потом поменяй монитор:
```bash
bash <(curl -s https://raw.githubusercontent.com/voxelpy/dotfiles/main/install.sh)
# После установки:
sed -i 's/HDMI-A-1/eDP-1/' ~/.config/hypr/hyprland.conf
```

---

## Что делает скрипт по шагам

1. Устанавливает `git`, `chezmoi`, `yay`
2. Скачивает все конфиги через chezmoi
3. Устанавливает все pacman пакеты из `pkglist.txt`
4. Устанавливает все AUR пакеты из `aurlist.txt`
5. Копирует Silent тему в `/usr/share/sddm/themes/`
6. Копирует `sddm.conf` в `/etc/`
7. Копирует `hyprland-sddm.conf` в `/etc/hypr/`
8. Копирует аватарку в `/usr/share/sddm/faces/`
9. Включает SDDM как дисплей менеджер
10. Создаёт папку для скриншотов

---

```bash
# Монитор

```

## Если что-то пошло не так

**Не запускается SDDM:**
```bash
sudo systemctl status sddm
journalctl -u sddm -n 50
```

**Waybar не отображается:**
```bash
waybar &
```

**Нет звука:**
```bash
pgrep swayosd-server || swayosd-server &
```

**Обои не восстановились:**
```bash
waypaper --restore
```

# 📖 Полный гайд по горячим клавишам Hyprland

> `Alt` = главный модификатор (`$mainMod`) — не Win/Super, а именно **Alt**

---

## 🖥️ Управление окнами

| Сочетание | Действие |
|-----------|----------|
| `Alt + C` | Закрыть активное окно |
| `Alt + V` | Плавающий режим (floating toggle) |
| `Ctrl + Alt + F` | Тоже плавающий режим (второй бинд) |
| `Super + F` | Полноэкранный режим |
| `Alt + M` | Выйти из Hyprland |

---

## 📐 Изменение размера окна

| Сочетание | Действие |
|-----------|----------|
| `Alt + L` | Расширить окно вправо |
| `Alt + H` | Сузить окно влево |
| `Alt + K` | Уменьшить окно по высоте (вверх) |
| `Alt + J` | Увеличить окно по высоте (вниз) |

---

## 🔀 Перемещение окон между позициями

| Сочетание | Действие |
|-----------|----------|
| `Alt + Shift + H` | Поменять окно с соседним слева |
| `Alt + Shift + J` | Поменять окно с соседним снизу |
| `Alt + Shift + K` | Поменять окно с соседним сверху |
| `Alt + Shift + L` | Поменять окно с соседним справа |

---

## 🖱️ Мышь

| Сочетание | Действие |
|-----------|----------|
| `Alt + ЛКМ` (зажать и тащить) | Перемещать окно |
| `Alt + ПКМ` (зажать и тащить) | Изменять размер окна |

---

## 🔍 Навигация по воркспейсам

| Сочетание | Действие |
|-----------|----------|
| `Alt + 1–0` | Перейти на воркспейс 1–10 |
| `Alt + Tab` | Вернуться на предыдущий воркспейс |
| `Alt + S` | Открыть/закрыть **скрэтчпад** (magic workspace) |

---

## 📦 Перемещение окна на воркспейс

| Сочетание | Действие |
|-----------|----------|
| `Alt + Shift + 1–0` | Отправить окно на воркспейс 1–10 (тихо, без переключения) |
| `Alt + Shift + S` | Отправить окно в скрэтчпад |

---

## 🎯 Фокус (переключение между окнами)

| Сочетание | Действие |
|-----------|----------|
| `Alt + ←` | Фокус на окно слева |
| `Alt + →` | Фокус на окно справа |
| `Alt + ↑` | Фокус на окно сверху |
| `Alt + ↓` | Фокус на окно снизу |

---

| Сочетание | Действие |
|-----------|----------|
| `Alt + Enter` | Открыть терминал **Kitty** |

## 📸 Скриншоты

| Сочетание | Действие |
|-----------|----------|
| `PrintScreen` | Скриншот выделенной области → сохранить в ~/Pictures |
| `Super + S` | Скриншот выделенной области (через grim+slurp) → сохранить |

| `Super + Ctrl + S` | Скриншот выделенной области → в **буфер обмена** |

---

## 🎬 Медиа (кнопки на клавиатуре)

| Сочетание | Действие |
|-----------|----------|
| `⏭` (Next) | Следующий трек (playerctl) |
| `⏸` (Pause) | Пауза/воспроизведение (playerctl) |
| `▶` (Play) | Пауза/воспроизведение (playerctl) |
| `⏮` (Prev) | Предыдущий трек (playerctl) |
| `Alt + Q` | Пауза/продолжение видео в **MPV** |

---

## 🔊 Громкость и яркость

| Сочетание | Действие |
|-----------|----------|
| `🔊` (Volume Up) | Увеличить громкость на 5% |
| `🔉` (Volume Down) | Уменьшить громкость на 5% |
| `🔇` (Mute) | Выключить/включить звук |
| `🎙️` (Mic Mute) | Выключить/включить микрофон |
| `☀️+` (Brightness Up) | Увеличить яркость на 10% |
| `☀️-` (Brightness Down) | Уменьшить яркость на 10% |
